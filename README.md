# AI Store — India's Premium AI Assistant Store

A modern, responsive, and feature-rich e-commerce website for premium AI assistants.

## 🚀 Features

- **Premium UI** – Glassmorphism, neon blue + purple theme, dark mode, smooth animations
- **Product Listing** – Single and combo packs (Zara, MAX, Jarvis)
- **Dynamic UPI Payment** – QR code generation using `api.qrserver.com`, UPI ID copy, and instant WhatsApp order submission
- **Floating Support** – WhatsApp and Telegram floating buttons
- **Responsive** – Mobile-first design, works on all screen sizes
- **Interactive Elements** – Cursor glow, particle background, scroll animations, FAQ accordion
- **SEO Friendly** – Clean HTML5, semantic tags, proper meta descriptions
